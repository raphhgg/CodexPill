# Accounts

CodexPill accounts are locally saved Codex authentication snapshots with user-facing metadata. Account features must preserve a clear distinction between adding an account and switching the active Codex account.

## Feature Inventory

### Add Account

Add another Codex account to CodexPill's local catalog without changing the currently active local Codex account.

See [Add Account](add-account.md).

### Switch Account

Activate a saved account on This Mac or on a configured remote host through the existing switch flows.

### Refresh Account Status

Refresh account metadata, plan, and rate-limit snapshots from Codex.

See [Refresh Accounts](05-refresh-accounts.md).

### Remote Account Setup

Install and switch saved account snapshots on configured remote hosts.
